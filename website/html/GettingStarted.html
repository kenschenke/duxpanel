Getting Started

If you're the ready, fire, aim type (aren't all programmers?), you should find everything you need on this page and the page on Panel Basics to start using DuxPanel.

Installation

    npm install duxpanel

You also need to link to the duxpanel.css file in the node_modules/duxpanel/website/dist/ directory. 

    <link rel="stylesheet" href="duxpanel.css"/>
You might be tempted to use the duxpanel.css file in your project with webpack and an import statement. This will work as long as you don't need to override any of the CSS in the file.

Browser Requirements

DuxPanel doesn't place any special requirements on browser support. It has been tested on the modern evergreen browsers (Chrome and Firefox) as well as Internet Explorer 10 and 11. It definitely won't work on Internet Explorer 8 or earlier.
